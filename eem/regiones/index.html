Crecimiento Regional en México



Para calcular las tasas de crecimiento por zona en el país, se puede hacer el siguiente indicador a partir de datos del INEGI.

Primero, se agrupan los estados en zonas. El peso de cada estado en su zona se calcula como el peso promedio (de su PIB) de los últimos 4 años en toda la region.

Ese peso se multiplica por el nivel del Indicador Trimestral de Actividad Económica (ITAE) en un periodo cero, aplicando desde ahí la misma tasa de crecimiento del ITAE de ese estado para obtener un ITAE ponderado por zona.

La suma de estos constituye un indicador a nivel zona, misma que nos permite encontrar una tasa de cambio anual.

Ejemplo

El estado 1 tiene el 75% del PIB generado en los últimos 4 años en la región a la que pertenece (es una de solo 2 estados):

estado_1 <- 0.75
estado_2 <- 0.25

Obtenemos el ITAE de ambos estados y por consecuente, su crecimiento en los últimos 4 trimestres:

crecim_1 <- c(1.04, #crecimiento 4%
              1.02, #crecimiento 2%, etc..
              1.05,
              1.07)

crecim_2 <- c(1.10, 
              1.12,
              1.15,
              1.20)

Usando un trimestre (por ejemplo el n-4) como inicio de la serie, usamos la función crecer() en “inegiR”

library(inegiR)

# solo para demostrar la función:
crecer <- function (tasas, comienzo) 
{
    m <- tasas
    r <- 0:length(m)
    n <- comienzo
    for (i in 1:length(m)) {
        r[i] <- m[i] * n
        n <- r[i]
    }
    salida <- r[1:length(m)]
    return(salida)
}

# calculamos los indicadores en base a sus crecimientos
ind_1 <- crecer(tasas = crecim_1, 100 * estado_1)
ind_2 <- crecer(tasas = crecim_2, 100 * estado_2)

# ver indicadores la serie
ind_1
# [1] 78.00000 79.56000 83.53800 89.38566

Estos dos indicadores representan el PIB (aunque en índice) que puede ser sumado para obtener un indicador a nivel región. Esto se hace finalmente mediante una simple suma:

region <- ind_1 + ind_2

region
# [1] 105.5000 110.3600 118.9580 131.8897

La tasa de crecimiento de esa serie es la tasa de crecimiento de la región.
